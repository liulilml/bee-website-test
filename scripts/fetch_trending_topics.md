# 热点选题自动抓取 - 设计文档

## 目标
每天定时从多个平台拉取热点，筛选与"AI工具/编程/效率"相关的话题，
存入飞书多维表格"选题池"，供 LEE 和 Cobalt 讨论选题。

## 架构

```
定时任务（每天 9:00 / 18:00）
    ↓
拉取热搜（微博/知乎/B站/百度）
    ↓
AI 筛选（关键词匹配 + 相关性评分）
    ↓
去重（与已有选题对比）
    ↓
写入多维表格"选题池"
    ↓
通知 LEE（飞书消息）
```

## 选题角度模板
- 工具安装和使用教程
- 踩坑经验和解决方案
- 对比评测（A vs B）
- 新手入门指南
- 实战案例分享
- 热点事件解读

## 评分规则
- 热度分（0-5）：基于平台热度值
- 相关性（0-5）：与我们账号定位的匹配度
- 时效性（0-5）：话题的新鲜程度
- 总分 = 热度 + 相关性*2 + 时效性

## 定时频率
- 每天 9:00：早间热点扫描
- 每天 18:00：下午热点扫描
