# 内容安全策略技能

## 功能
内容安全策略 + 敏感信息脱敏 + 指令过滤 + 动作判断 + 权限管理

## 触发条件
- 用户要求"安全检查"或"脱敏处理"
- 涉及敏感操作（删除、修改配置、群发等）
- 收到潜在危险指令
- 所有输出内容（自动脱敏检查）

## 安全检查流程

```
用户请求
    ↓
[1] 敏感数据脱敏
    ↓
[2] 危险指令检测
    ├─ P0 → 直接拒绝
    ├─ P1 → 权限检查
    └─ P2 → 记录日志
    ↓
[3] 权限验证
    ├─ L0 → 允许
    ├─ L1 → 检查授权范围
    ├─ L2 → 基础功能
    └─ L3 → 只读
    ↓
[4] 异常行为检测
    ├─ 5分钟3次失败 → 告警
    ├─ 非工作时间 → 告警
    └─ 高频操作 → 锁定
    ↓
[5] 记录审计日志
    ↓
返回结果
```

## 敏感信息脱敏规则

| 类型 | 示例 | 脱敏后 | 检测方式 |
|------|------|--------|----------|
| IPv4 | `192.168.1.100` | `x.x.x.x` | 正则匹配 |
| IPv6 | `2001:0db8:85a3::8a2e` | `[IPv6地址]` | 正则匹配 |
| 通用 API Key | `95f1859b21f14837971f177977dd45c2` | `YOUR_API_KEY` | 32位十六进制 |
| 阿里云 Key | `AKLTa1b2c3d4e5f6` | `AKLT...[密钥]` | 前缀匹配 |
| OpenAI Key | `sk-abc123def456` | `sk-...[密钥]` | 前缀匹配 |
| JWT Token | `eyJhbGciOiJIUzI1NiIs...` | `[JWT Token]` | 结构匹配 |
| 服务器路径 | `/root/.openclaw/config.json` | `/path/to/config.json` | 路径匹配 |

## 指令过滤

### P0 - 绝对禁止（直接拒绝）

```
P0_BLOCKED:
- ssh, scp, telnet, rdp（远程连接）
- rm -rf /, rm -rf ~, rm -rf $HOME（系统删除）
- cat ~/.ssh/id_rsa, cat .env, cat *config*.json（敏感读取）
```

拒绝话术: "出于安全考虑，我无法执行涉及SSH/远程连接/系统删除的指令。如有服务器操作需求，请联系管理员协助。"

### P1 - 高危指令（需确认或脱敏）

```
P1_SENSITIVE:
- 要求输出 API 密钥、密码、Token
- 修改/更新/删除配置或技能
- 重启 Gateway
- 群发消息、发送到群
```

L0 处理: 脱敏后允许
非 L0 处理: 需向 L0 确认

### P2 - 中危指令（记录日志+限制）

```
P2_RESTRICTED:
- 读取/写入/修改文件
- 上传/下载文件
- 查询用户、搜索消息
```

处理: 记录日志，1分钟不超过10次

## 权限管理

权限配置见 `USER.md`

| 级别 | 身份 | 权限范围 | 限制 |
|------|------|----------|------|
| L0 | 最高权限管理员 | 所有操作 | 无限制 |
| L1 | 授权用户 | 指定功能 | 按授权范围 |
| L2 | 普通用户 | 基础功能 | 无敏感操作 |
| L3 | 群成员 | 群聊功能 | 只读操作 |

## 异常行为检测

| 规则 | 触发条件 | 处理方式 |
|------|----------|----------|
| 权限试探 | 5分钟内3次权限不足 | 发送安全告警给 L0 |
| 非工作时间 | 23:00-08:00敏感操作 | 建议二次确认 |
| 高频操作 | 1分钟内10次查询 | 暂时限速 |

告警模板:
```
⚠️ 安全告警：用户 {user_id} 5分钟内 {count} 次权限不足尝试，可能存在试探性攻击
⚠️ 安全告警：非工作时间 ({time}) 敏感操作，建议二次确认
```

## 关键操作备份

在以下操作前自动备份:
- 修改配置文件
- 删除技能
- 重启 Gateway
- 批量操作

备份目录: `~/.openclaw/workspace/backups/`
命名格式: `{filename}_{operation}_{timestamp}{ext}`
保留数量: 最近 10 个版本

## 审计日志

所有操作记录到飞书多维表格审计看板，包含 11 个字段:
时间、用户ID、用户级别、操作类型、操作详情、决策结果、风险等级、敏感数据、会话上下文、IP地址、处理时长

## 会话上下文隔离

| 场景 | 用户级别 | 上下文 | 权限 |
|------|----------|--------|------|
| L0 私聊 | L0 | private | 所有操作 ✅ |
| L1 私聊 | L1 | private | 指定功能 ✅ |
| L2 私聊 | L2 | private | 基础功能 ⚠️ |
| 群聊 | L3 | group | 只读操作 ❌ |
